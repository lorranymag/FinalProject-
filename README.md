## How to use the application
 Open the project in Visual Studio Code  
1. Navigate to the `src` folder  
2. Locate the main file (`App.java and ToDoList.java`)  
3. Run the program using the IDE or terminal:

### Basic Functionality
This application allows the user to add simple tasks to a to-do list.

1. Run the program
2. A window will appear with a text field and a button
3. Type a task into the text field
4. Click the "Add" button
5. The task will be displayed on the screen
